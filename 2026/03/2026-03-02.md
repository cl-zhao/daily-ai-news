## 🇨🇳 国内大模型动态

### 智谱 GLM-5 技术细节全公开

智谱AI发布了GLM-5的完整技术报告，这是目前国内参数量最大的开源大模型之一：

- **架构**：744B MoE（混合专家）架构，每次推理激活40B参数
- **训练规模**：28.5T tokens 训练数据
- **核心创新**：集成 DeepSeek 稀疏注意力机制（DSA），大幅降低推理成本
- **国产适配**：已适配华为昇腾等7大国产芯片
- **定位**：推动编程范式从"氛围编程"转向"智能体工程"

**技术亮点**：DSA机制根据Token重要性动态分配注意力资源，在不损失长上下文能力的前提下，算力开销大幅削减。

### MiniMax M2.5：高性价比之选

- **定位**：对标 Claude Opus 4.6，但成本仅为其1/10
- **用户规模**：已获得约Claude三分之一的用户基数
- **适用场景**：代码生成、智能体任务、音视频内容生成

### 阿里通义千问 Qwen 3.5

- 推出首个原生多模态大模型
- 统一架构整合语言推理与视觉感知
- 在效果、成本与多模态理解深度上同时超越 Qwen3-Max

## 🌍 国际大模型动态

### Google Gemini 3.1 Pro Preview

- **发布时间**：2026年2月19日
- **性能提升**：ARC-AGI-2得分77.1%，是Gemini 3 Pro的两倍多
- **核心特性**：
  - 1M token 上下文窗口
  - 原生视频和音频理解
  - Google Search 接地增强
- **注意**：Gemini 3 Pro Preview 将于3月9日下线

### Anthropic Claude 系列更新

- **Claude Sonnet 4.6**：新中端模型，定价与4.5相同，免费用户自动升级
- **Claude Opus 4.6**：METR认证拥有最长自主任务能力（14.5小时@50%）
- **Claude Code Security**：新代码安全审查功能

### OpenAI ChatGPT 更新

- **上下文扩展**：Thinking模式下总上下文扩展至256K（128K输入+128K输出）
- **Projects功能**：支持Slack/Drive链接，响应可保存为可复用源
- **文件上传**：每条消息从10个增加到20个
- **Codex macOS**：桌面端多代码智能体并行管理工具

### xAI Grok 4.20 Beta

- **创新点**："快速学习"架构，每周根据用户反馈自动改进——这是首个具备此能力的前沿模型
- **新功能**：医疗文档图片分析、工程推理增强、4智能体并行协作

### Perplexity Computer

- **定位**：全新自主智能体工具
- **特点**：同时使用19个AI模型，为复杂工作流创建子智能体
- **定价**：仅限Max订阅用户（$200/月）
- **战略转变**：完全放弃广告业务，专注企业订阅和研究用户
